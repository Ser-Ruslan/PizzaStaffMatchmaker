Цель проекта: Создать эффективный, удобный и масштабируемый веб-сервис для автоматизации процессов найма персонала для всех пиццерий сети. Сервис должен упростить подачу заявок для кандидатов и управление вакансиями и кандидатами для HR-отдела и менеджеров пиццерий.

Целевая аудитория:

Кандидаты (Соискатели): Люди, ищущие работу на позициях: повар-пиццайоло, кассир, курьер, менеджер смены, управляющий пиццерией и т.д.
HR-менеджеры сети: Сотрудники, отвечающие за централизованный подбор персонала, публикацию вакансий, первичный отбор.
Менеджеры/Управляющие отдельных пиццерий: Сотрудники, участвующие в отборе кандидатов для своей конкретной точки, проводящие собеседования.
Администраторы системы: Технические специалисты, поддерживающие работоспособность сервиса.
Основные роли пользователей и их права:

Кандидат (Анонимный / Зарегистрированный):
Просмотр списка открытых вакансий с фильтрацией (по городу, по типу должности, по конкретной пиццерии).
Просмотр детальной информации о вакансии.
Регистрация/Авторизация (через email/пароль, возможно OAuth с Google/VK).
Создание и редактирование своего профиля/резюме (контактная информация, опыт работы, образование, желаемая должность, график, предпочтительные пиццерии, загрузка фото и файла резюме).
Отклик на вакансии (с возможностью прикрепить сопроводительное письмо).
Просмотр статуса своих откликов.
Получение уведомлений (о новых подходящих вакансиях, об изменении статуса отклика).
HR-Менеджер Сети:
Все права Кандидата (для тестирования).
Управление вакансиями: создание, редактирование, публикация, архивация вакансий. Возможность указать пиццерию (или несколько), для которой открыта вакансия.
Просмотр списка всех откликнувшихся кандидатов с возможностью фильтрации и поиска.
Просмотр профилей кандидатов и их резюме.
Изменение статуса откликов (Новый, Рассмотрение, Приглашен на собеседование, Отказ, Принят на работу и т.д.).
Добавление комментариев к кандидатам/откликам (видимых только для HR и менеджеров пиццерий).
Назначение собеседований (с указанием даты, времени, места/формата и ответственного менеджера пиццерии).
Коммуникация с кандидатами через систему (или шаблоны email-уведомлений).
Просмотр статистики по вакансиям и кандидатам.
Управление списком пиццерий (добавление, редактирование информации о них).
Менеджер Пиццерии:
Просмотр вакансий, открытых для его пиццерии.
Просмотр кандидатов, откликнувшихся на вакансии его пиццерии или переданных ему HR-менеджером.
Просмотр профилей этих кандидатов.
Участие в изменении статуса откликов (например, "Одобрен менеджером", "Пригласить на повторное собеседование").
Добавление комментариев к кандидатам/откликам.
Просмотр назначенных ему собеседований.
Получение уведомлений о новых откликах на вакансии его пиццерии.
Администратор Системы:
Управление учетными записями всех пользователей (создание, блокировка, изменение ролей HR и Менеджеров).
Доступ к Django Admin панели для управления всеми моделями данных.
Настройка системных параметров (например, шаблоны уведомлений, категории должностей).
Просмотр логов системы.
Ключевые функции (MVP - Minimum Viable Product):

Для Кандидатов:
Регистрация и авторизация.
Создание/редактирование профиля с базовыми полями (ФИО, контакты, город, желаемая должность, коротко о себе).
Загрузка файла резюме (PDF, DOC, DOCX).
Просмотр списка вакансий с фильтрацией по городу и типу должности.
Отклик на вакансию.
Просмотр статуса своих откликов.
Для HR-Менеджера и Менеджера Пиццерии:
Авторизация.
Создание/редактирование/публикация вакансий (с указанием города, пиццерии, описания, требований, условий).
Просмотр списка откликов на вакансии.
Просмотр профилей кандидатов и их резюме.
Изменение статуса отклика (упрощенный набор статусов для MVP).
Базовые комментарии к кандидатам.
Административная панель Django:
Управление пользователями, вакансиями, откликами, пиццериями, типами должностей.
